Данный проек реализует поставленную задачу:

Требуется спроектировать систему распространения сообщений со шпионами. Система состоит из узлов, 
связанных между собой в одностороннем порядке (орграф). Каждый из узлов может получать сообщение, 
при этом запоминает его если сообщение обычное, не запоминает если оно секретное, но в любом случае 
передаёт сообщение дальше, всем своим "связным". В сети могут присутствовать "шпионские узлы". 
Шпионский узел ведёт себя также как обычный узел, но получив секретное сообщение, дополнительно
к базовому функционалу, сбрасывает это секретное сообщение в "нычку". Нычка - одна на всех шпионов, 
причём о том, где лежит нычка, шпион узнаёт ещё до заселения в сеть передачи сообщений (иными словами, при конструировании).
В качестве бд использовал SQLite

Привер Входных данных:
8                          - количесвто сообщений
Messages:		   - кодовое слово для обозначения начала сообщений
#0 Hello friends !	   ----- сообщения начинаются с символа #k
#0 how are you ?	   ----- где k - это число из {0, 1}, если k =
#1 COME HOME		   ----- 0 - значит сообщение не секретное и доступно
#1 TAKE ICE CREAM	   ----- всем пользователям сети
#0 I saw a great movie	   ----- 1 - значит сообщение СЕКРЕТНОЕ и будет 
#1 GO HERE AND WATCH A MOVIE---- прочитано только шпионами
#1 FASTLY!		   ----- 
#0 see it on the link : www.movie.com/123		   -----
6                          - количесвто пользователей в сети
People:		   	   - кодовое слово для обозначения начала списка пользователей
#0 Jack 2 Luke Oliver      ----- каждый пользователь описан строкой типа (#k name m name1, name2, .. nameM)
#0 Luke 1 Ella		   ----- где k - это число из {0, 1}, если k =
#1 Ella 0		   ----- 0 - значит это обычный пользователь
#1 Oliver 2 Henry Grace	   ----- 1 - значит это секретный агент(шпион)
#0 Henry 0		   ----- m - количесвто его друзей в сети
#0 Grace 0		   ----- name1, name2, ... nameM - это имена друзей пользователья name

После исполнения программы: 
-- в базу данных для всех пользователей "openstream.db" будут помещены все сообщения отмеченные #0
-- в секрутную базу данных(нычка) "secret.db" , доступную только для шпионов, будут помещены все сообщения отмеченные #1
Приятного пользования! P.S. после каждого запуска, все базы очищаются
